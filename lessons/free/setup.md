# Setup: Supabase and Firebase

In this lesson, you will provision Supabase (for Postgres + pgvector) and Firebase (Auth + Functions + Firestore).

Steps:
- Create a Supabase project and run `allie/sql/schema.sql` in the SQL Editor.
- Create a Firebase project, enable Authentication and Firestore.
- Deploy functions in `firebase/functions` and set Stripe keys.
