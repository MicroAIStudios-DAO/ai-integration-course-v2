# Archived Documentation

This folder holds **historical, point-in-time documents** — deployment status
snapshots, one-off fix summaries, and generated reports from earlier in the
project's history. They are kept for reference and provenance but are **no longer
authoritative**.

For current documentation, start at the [documentation index](../README.md) or
the [root README](../../README.md).

> ⚠️ Links inside these archived files may point at paths that have since moved.
> Treat their contents as a snapshot of when they were written.
